Outline 客户端 1.15.0 及以上版本。

本教程提供了详细的操作步骤，可帮助您实现 Shadowsocks-over-WebSocket，这是一种强大的技术，可在常规 Shadowsocks 连接被屏蔽的环境中绕过审查。通过将 Shadowsocks 流量封装在 WebSocket 中，您可以将其伪装成标准 Web 流量，从而提高弹性和可访问性。

注意：只有 Outline 客户端 1.15.0 及以上版本支持 Shadowsocks-over-WebSocket。您必须保留现有配置，才能支持较低的客户端版本。

第 2 步：公开 Web 服务器

若要让 WebSocket Web 服务器可公开访问，您需要将其公开到互联网并配置 TLS。您可以通过多种方式实现此目的。您可以使用 Caddy、nginx 或 Apache 等本地 Web 服务器，确保它具有有效的 TLS 证书，也可以使用 Cloudflare Tunnel 或 ngrok 等隧道服务。

第 4 步：连接到 Outline 客户端

使用某个官方 Outline 客户端应用（1.15.0 及更高版本），将新创建的动态访问密钥添加为服务器条目。点击连接，开始使用 Shadowsocks-over-WebSocket 配置通过隧道连接到服务器。
