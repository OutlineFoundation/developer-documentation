本指南详细介绍了如何设置 Go 环境以及在 Go 代码中使用 Outline SDK。

我们将创建一个名为 splitfetch 的示例应用以展示 SDK 的一项功能。此应用会提取一个网页，然后使用 Outline SDK 将初始 TCP 数据流拆分成两个独立的数据包，而非以单个网络数据包发送请求。这有助于绕过某些形式的网络干预。

您可以在 Linux、Mac 和 Windows 上运行此应用。如需与移动应用集成，请参阅将 Outline SDK 添加到移动应用。
