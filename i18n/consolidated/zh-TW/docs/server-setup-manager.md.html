本指南將逐步引導您設定 Outline 伺服器，讓您可以安全且不受限制地存取網際網路。

逐步說明

  1. 從 https://getoutline.org 下載 Linux、macOS 或 Windows 專用的 Outline Manager。

  2. 在 Outline Manager 中點選「+」按鈕以新增伺服器。

  3. 選擇偏好的雲端服務供應商 (例如 DigitalOcean、Amazon、Google Cloud)；若您有自己的伺服器，則可以點選「隨時隨地設定 Outline」。

  4. 按照畫面上的指示操作，指示內容會根據您選擇的供應商或自行設定有所不同。

  5. Outline Manager 會逐步引導您建立伺服器、安裝 Outline 並產生存取金鑰。
