คู่มือนี้จะอธิบายกระบวนการตั้งค่าเซิร์ฟเวอร์ Outline เพื่อการเข้าถึงอินเทอร์เน็ตที่ปลอดภัยและไม่จำกัด

  1. ดาวน์โหลด Outline Manager สำหรับ Linux, macOS หรือ Windows จาก https://getoutline.org

  2. ใน Outline Manager ให้คลิกปุ่ม "+" เพื่อเพิ่มเซิร์ฟเวอร์ใหม่

  3. เลือกผู้ให้บริการคลาวด์ที่ต้องการ (เช่น DigitalOcean, Amazon, Google Cloud) หรือ "ติดตั้ง Outline ได้ทุกที่" หากมีเซิร์ฟเวอร์ของตนเอง

  4. ทำตามวิธีการบนหน้าจอสำหรับผู้ให้บริการที่เลือกหรือติดตั้งด้วยตนเอง

  5. Outline Manager จะแนะนำขั้นตอนการสร้างเซิร์ฟเวอร์ให้คุณ

ขั้นตอนถัดไป

เมื่อตั้งค่าเซิร์ฟเวอร์ Outline แล้ว คุณก็พร้อมแชร์การเข้าถึงให้กับผู้ใช้ได้เลย
