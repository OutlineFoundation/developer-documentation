Deze handleiding leidt je door het proces om een Outline-server in te stellen zodat je mensen beveiligde, onbeperkte internettoegang kunt geven.

Stapsgewijze instructies

  1. Download Outline Manager voor Linux, macOS of Windows via https://getoutline.org.

  2. Klik in Outline Manager op het plusje (+) om een nieuwe server toe te voegen.

  3. Kies een cloudprovider (zoals DigitalOcean, Amazon, Google Cloud). Of kies 'Outline overal installeren' als je een eigen server hebt.

  4. Volg de instructies op het scherm voor de gekozen provider of zelfinstallatie.

  5. Outline Manager leidt je door het proces om een server te maken, Outline te installeren en toegangssleutels te genereren.
