หลังจากติดตั้งเซิร์ฟเวอร์ Outline เรียบร้อยแล้ว ระบบจะนำคุณไปยังหน้ารายละเอียดของเซิร์ฟเวอร์ใน Outline Manager ซึ่งหน้านี้จะเป็นฮับศูนย์กลางสำหรับการสร้างและจัดการสิทธิ์เข้าถึงเซิร์ฟเวอร์ที่ใช้[คีย์การเข้าถึง](../../concepts#access-keys)

## สร้างและแชร์คีย์การเข้าถึง {#create_and_share_access_keys}

- **สร้างคีย์ใหม่:** คลิกไอคอน **+** เล็กๆ ที่ด้านล่างของหน้าเพื่อสร้างคีย์ใหม่

- **แชร์คีย์ใหม่:** คลิกไอคอนแชร์ที่อยู่ด้านขวาของแต่ละคีย์เพื่อสร้างคำเชิญ จากนั้นคัดลอกคำเชิญและแจกจ่ายไปยังผู้ใช้ที่ต้องการผ่านแพลตฟอร์มการรับส่งข้อความที่ต้องการ

### กระบวนการเชิญผู้ใช้ {#user_invitation_process}

เมื่อได้รับคำเชิญ ผู้ใช้จะต้องทำตามขั้นตอนเหล่านี้เพื่อเชื่อมต่อกับเซิร์ฟเวอร์ Outline

1. ผู้ใช้คลิกลิงก์การติดตั้งที่มีให้ในอุปกรณ์ของตน

2. หากยังไม่ได้ติดตั้งแอป Outline ผู้ใช้จะได้รับข้อความแจ้งให้ดาวน์โหลดและติดตั้งแอป

3. แอป Outline จะส่งข้อความโดยอัตโนมัติเพื่อแจ้งให้ผู้ใช้เพิ่มเซิร์ฟเวอร์โดยใช้คีย์การเข้าถึงที่ไม่ซ้ำกันที่ฝังอยู่ในลิงก์คำเชิญ

4. ในตอนนี้ผู้ใช้สามารถใช้ปุ่ม**เชื่อมต่อ**และ**เลิกเชื่อมต่อ**ภายในแอป Outline เพื่อควบคุมการเข้าถึงอินเทอร์เน็ตผ่านเซิร์ฟเวอร์ Outline ได้

## เพิกถอนสิทธิ์เข้าถึงเซิร์ฟเวอร์ {#revoke_access_to_your_server}

การจัดการสิทธิ์เข้าถึงของผู้ใช้เป็นสิ่งสำคัญ Outline Manager มีวิธีการง่ายๆ ในการเพิกถอนสิทธิ์เข้าถึงเมื่อจำเป็น

- **ค้นหาคีย์:** ไปยังหน้ารายละเอียดเซิร์ฟเวอร์ใน Outline Manager แล้วระบุคีย์การเข้าถึงที่เกี่ยวข้องกับผู้ใช้ที่มีสิทธิ์เข้าถึงที่ต้องการเพิกถอน

- **ลบคีย์:** คุณจะเห็นไอคอนเมนูทางด้านขวาของคีย์ ให้คลิกไอคอนดังกล่าวแล้วเลือก**นำออก**
