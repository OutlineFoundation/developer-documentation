Outline SDK เป็นไลบรารีและคอลเล็กชันเครื่องมือที่ออกแบบมาเพื่อช่วยนักพัฒนาสร้างแอปพลิเคชันที่ป้องกันการรบกวนและการเซ็นเซอร์ระดับเครือข่าย

## คุณใช้ Outline SDK ทำอะไรได้บ้าง {#what_can_you_do_with_the_outline_sdk}

- **สร้างเครื่องมือเพื่อช่วยลดการรบกวนระดับเครือข่าย**\

สร้างเครื่องมือใหม่เพื่อปกป้องผู้ใช้ไม่ให้ถูกรบกวนระดับเครือข่ายและเข้าถึงอินเทอร์เน็ตที่เปิดกว้างได้อย่างเสถียร

- **เพิ่มการป้องกันให้แอปที่มีอยู่**\

ผสานรวมการป้องกันการรบกวนระดับเครือข่ายลงในแอปที่มีอยู่โดยตรง เช่น แอปการสื่อสารหรือเนื้อหา ซึ่งจะช่วยให้แอปมีความยืดหยุ่นต่อการบล็อกมากขึ้น

- **แก้ปัญหาและตรวจวัดการรบกวน**\

ใช้เครื่องมือบรรทัดคำสั่งที่มีให้เพื่อวินิจฉัยปัญหาการเชื่อมต่อ ตรวจวัดการรบกวนระดับเครือข่าย และทดสอบกลยุทธ์การหลบเลี่ยงต่างๆ

## ข้อดีหลัก {#key_advantages}

- **มัลติแพลตฟอร์ม**\

ทำงานได้ในระบบปฏิบัติการหลักๆ อย่าง Android, iOS, Windows, macOS และ Linux.

- **เทคโนโลยีที่ได้รับการพิสูจน์แล้ว**\

เทคโนโลยีนี้ได้รับการทดสอบผ่านการใช้งานจริง ซึ่งขับเคลื่อนไคลเอ็นต์ Outline และ Intra ที่ใช้ในวงกว้าง ตลอดจนแอปหลายแอปที่ช่วยให้ผู้ใช้หลายล้านคนเข้าถึงอินเทอร์เน็ตภายใต้สภาวะที่ไม่เอื้ออำนวยได้

- **ประกอบกันได้**\

ออกแบบมาสำหรับการแยกย่อยแบบโมดูลและคอมโพเนนต์ที่นำกลับมาใช้ใหม่ได้ จึงช่วยให้นักพัฒนาสร้างสรรค์กลยุทธ์แบบกำหนดเองได้
