---
title: "Deploy Using the Outline Manager"
sidebar_label: "Using Outline Manager"
---

Ce guide vous explique comment configurer un serveur Outline pour fournir un accès sécurisé et sans restriction à Internet.

1. Téléchargez Outline Manager pour Linux, macOS ou Windows depuis la page <https://getoutline.org>.

2. Dans Outline Manager, cliquez sur le bouton + pour ajouter un serveur.

3. Choisissez le fournisseur de services cloud de votre choix (par exemple, DigitalOcean, Amazon ou Google Cloud) ou cliquez sur "Installer Outline sur n'importe quel serveur cloud" si vous avez votre propre serveur.

4. Suivez les instructions à l'écran spécifiques au fournisseur que vous avez choisi ou celles de votre propre serveur.

5. Outline Manager vous indiquera comment créer un serveur.

## Étape suivante

Maintenant que votre serveur Outline est configuré, vous pouvez partager l'accès avec vos utilisateurs.
