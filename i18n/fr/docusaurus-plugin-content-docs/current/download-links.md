Vous pouvez télécharger Outline Manager et le client Outline à partir des liens ci-dessous. Pour en savoir plus sur la configuration d'Outline, consultez la page <https://getoutline.org/>.


:::warning[Important]
Les liens ci-dessous sont les seules sources officielles pour télécharger Outline Manager et le client Outline. Nous ne partageons pas directement ces fichiers sur les réseaux sociaux ni dans les applications de chat, comme Telegram et WhatsApp.
:::
