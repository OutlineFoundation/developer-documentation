title: "Deploy Using the Outline Manager"
sidebar_label: "Using Outline Manager"
---

Questa guida illustra il processo di configurazione di un server Outline per fornire un accesso a internet sicuro e senza limitazioni.

1. Scarica Outline Manager per Linux, macOS o Windows da <https://getoutline.org>.

2. In Outline Manager, fai clic sul pulsante "+" per aggiungere un nuovo server.

3. Scegli il tuo cloud provider preferito (ad esempio DigitalOcean, Amazon, Google
Cloud) o "Configura Outline ovunque" se hai un tuo server.

4. Segui le istruzioni sullo schermo specifiche per il provider scelto o configuralo autonomamente.

5. Outline Manager ti guiderà nella creazione di un server.

## Passaggi successivi

Ora che hai configurato il server Outline, puoi condividere l'accesso con i tuoi utenti.
