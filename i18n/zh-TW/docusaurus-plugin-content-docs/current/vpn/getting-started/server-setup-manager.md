---
title: "Deploy Using the Outline Manager"
sidebar_label: "Using Outline Manager"
---

本指南將逐步引導您設定 Outline 伺服器，讓您享有安全且不受限制的上網體驗。

1. 

從 <https://getoutline.org> 下載 Linux、macOS 或 Windows 專用的 Outline Manager。

2. 

在 Outline Manager 中點選「+」按鈕以新增伺服器。

3. 

選擇偏好的雲端服務供應商 (例如 DigitalOcean、Amazon、Google Cloud)；若您有自己的伺服器，則可以點選「隨時隨地設定 Outline」。

4. 

按照畫面上的指示，完成所選供應商或自行設定的步驟。

5. 

Outline Manager 會逐步引導您建立伺服器。

## 後續步驟

設定好 Outline 伺服器後，即可將存取權分享給其他使用者。
