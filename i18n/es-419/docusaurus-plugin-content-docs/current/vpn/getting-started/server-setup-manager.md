---
title: "Deploy Using the Outline Manager"
sidebar_label: "Using Outline Manager"
---

En esta guía, se explica el proceso para configurar un servidor de Outline para
proporcionar acceso a Internet seguro y sin restricciones.

1. Descarga Outline Manager para Linux, macOS o Windows desde
<https://getoutline.org>.

2. En Outline Manager, haz clic en el botón "+" para agregar un servidor nuevo.

3. Elige tu proveedor de servicios en la nube (p. ej., DigitalOcean, Amazon o Google
Cloud) o "Configurar Outline en cualquier lugar" si tienes tu propio servidor.

4. Sigue las instrucciones en pantalla específicas para tu proveedor o
modo de autoconfiguración.

5. Outline Manager te guiará a través de la creación de un servidor.

## Próximos pasos

Ahora que tu servidor de Outline está configurado, ya puedes compartir el acceso con tus
usuarios.
