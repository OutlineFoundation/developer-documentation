---
title: "Deploy Using the Outline Manager"
sidebar_label: "Using Outline Manager"
---

Bu kılavuzda, internet erişimini güvenli ve sınırsız hale getirmek için Outline sunucusu kurma işlemi anlatılmaktadır.

1. [https://getoutline.org/tr/](https://getoutline.org) adresinden Outline Manager'ın Linux, macOS veya Windows sürümünü indirin.

2. Outline Manager'da "+" düğmesini tıklayıp yeni bir sunucu ekleyin.

3. Tercih ettiğiniz bulut sağlayıcıyı (ör. DigitalOcean, Amazon, Google Cloud) seçin ya da kendi sunucunuz varsa "Outline'ı dilediğiniz yere kurun"u tıklayın.

4. Seçtiğiniz sağlayıcıya veya kendi kurulumunuza göre ekrandaki talimatları uygulayın.

5. Outline Manager, sunucu oluşturma konusunda sizi yönlendirir.

## Sonraki adımlar

Outline sunucunuz ayarlandıktan sonra, erişimi kullanıcılarınızla paylaşabilirsiniz.
