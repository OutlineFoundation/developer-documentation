---
title: "Deploy Using the Outline Manager"
sidebar_label: "Using Outline Manager"
---

本指南详细介绍了如何设置 Outline 服务器，以便安全而不受限地访问互联网。

1. 

从 <https://getoutline.org> 下载适用于 Linux、macOS 或 Windows 的 Outline 管理器。

2. 

在 Outline 管理器中点击“+”按钮，以添加新服务器。

3. 

选择首选云服务提供商（例如 DigitalOcean、Amazon、Google Cloud）；如果您有自己的服务器，则选择“随时随地安装 Outline”。

4. 

按照屏幕上针对所选提供商的说明或自助安装说明操作。

5. 

Outline 管理器将全程引导您创建服务器。

## 后续步骤

您的 Outline 服务器已设置完毕，现已可以和用户分享访问权限。
