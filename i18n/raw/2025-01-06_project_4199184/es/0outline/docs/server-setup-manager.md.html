Esta guía explica el proceso de configurar un servidor de Outline para ofrecer acceso a Internet seguro y sin restricciones.

Instrucciones paso a paso

  1. Descarga Administrador de Outline para Linux, macOS o Windows enhttps://getoutline.org.

  2. En Administrador de Outline, haz clic en el botón más "+" para añadir un servidor nuevo.

  3. Elige el proveedor de servicios en la nube que prefieras (por ejemplo, DigitalOcean, Amazon o Google Cloud) o "Instala Outline en cualquier lugar" si tienes un servidor propio.

  4. Sigue las instrucciones que aparecen en pantalla específicas del proveedor que has elegido o configúralo por tu cuenta.

  5. Administrador de Outline te guiará por el proceso de crear un servidor, instalar Outline y generar claves de acceso.
