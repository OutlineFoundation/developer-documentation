  1. Si el cliente de Outline se crea a través de side-loading (es decir, descargando directamente el archivo APK e instalándolo), la app no recibirá actualizaciones automáticas y deberá actualizarse de forma manual para aprovechar las funciones más recientes. Para instalar la app, es posible que debas cambiar tus parámetros de configuración para permitir la instalación de apps desconocidas. Después de instalar Outline, se recomienda cambiar inmediatamente el parámetro de configuración para que se inhabilite la instalación de esas apps. Así, evitarás que se instalen apps potencialmente dañinas o maliciosas de fuentes no seguras. ↩
