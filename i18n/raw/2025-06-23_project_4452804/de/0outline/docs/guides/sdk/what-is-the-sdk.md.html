Das Outline SDK ist eine Bibliothek und eine Sammlung von Tools, die Entwicklern helfen sollen, Anwendungen zu erstellen, die vor Eingriffen auf Netzwerkebene und Zensur schützen.

Wofür kann das Outline SDK verwendet werden?

  • Entwicklung von Tools zur Abwehr von Eingriffen in Netzwerke
    Entwickeln Sie neue Tools zum Schutz der Nutzer vor Eingriffen auf Netzwerkebene und für einen zuverlässigen Zugang zum offenen Internet.

  • Zusätzlicher Schutz für bestehende Apps
    Binden Sie den Schutz vor Eingriffen auf Netzwerkebene direkt in bestehende Anwendungen wie Kommunikations- oder Content-Apps ein und machen Sie diese robuster gegen Blockierungen.

  • Fehlerbehebung und Messung von Eingriffen
    Mit den enthaltenen Befehlszeilentools können Sie Verbindungsprobleme diagnostizieren, Eingriffe in Netzwerke messen und verschiedene Umgehungsstrategien testen.

Wichtigste Vorteile

  • Plattformübergreifende Nutzung
    Das Outline SDK funktioniert mit allen gängigen Betriebssystemen, einschließlich Android, iOS, Windows, macOS und Linux.

  • Bewährte Technologie
    Die Technologie ist praxiserprobt und bildet die Grundlage für den weit verbreiteten Outline-Client und Intra sowie für Dutzende von Anwendungen, die Millionen von Endnutzern unter schwierigen Bedingungen den Zugang zum Internet ermöglichen.

  • Komponierbarkeit
    Das Outline SDK wurde für Modularität und wiederverwendbare Komponenten entwickelt. So können Entwickler individuelle Strategien umsetzen.
