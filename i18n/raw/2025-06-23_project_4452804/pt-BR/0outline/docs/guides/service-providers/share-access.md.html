Depois de instalar seu servidor do Outline, a página de detalhes do servidor dentro do Outline Manager será aberta. Essa página é sua central para gerar e gerenciar o acesso ao seu servidor usando chaves de acesso.

Criar e compartilhar chaves de acesso

  • Criar uma nova chave: clique no pequeno ícone de + na parte inferior da página para criar novas chaves.
  • Compartilhar a nova chave: à direita de cada chave, há um ícone de compartilhamento que você pode clicar para obter um convite. Copie o convite e envie-o para o usuário pela plataforma de mensagens que você quiser.
Dica: use uma plataforma de mensagens segura, como um e-mail criptografado ou um aplicativo de chat seguro, ao compartilhar os links do convite. Evite usar canais não seguros. Isso pode comprometer a chave de acesso.

Processo de convite de usuários

Ao receber o convite, o usuário deverá seguir estas etapas para se conectar ao seu servidor do Outline:

  1. Em um dispositivo, o usuário clica no link de instalação fornecido.
  2. Se o aplicativo Outline não estiver instalado, o usuário deverá fazer o download e instalar.
  3. O aplicativo Outline pede automaticamente que o usuário adicione seu servidor usando a chave de acesso exclusiva incorporada no link do convite.
  4. Agora o usuário pode usar os botões Conectar e Desconectar no aplicativo Outline para controlar o acesso à internet pelo servidor que você disponibilizou.

Revogar o acesso ao seu servidor

Gerenciar o acesso de usuários é essencial. O Outline Manager oferece um método simples e direto para revogar acessos quando necessário.

  • Localize a chave: vá até a página de detalhes do servidor no Outline Manager e identifique a chave de acesso correspondente ao usuário que terá o acesso revogado.
  • Exclua a chave: à direita da chave, há um ícone de menu. Clique nele e selecione Remover.
