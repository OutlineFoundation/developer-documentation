En esta guía, se explica el proceso para configurar un servidor de Outline para proporcionar acceso a Internet seguro y sin restricciones.

Requisitos previos

  • Un servidor (físico o virtual) que ejecute un sistema operativo compatible (Ubuntu 20.04 LTS o Debian 10)
  • Acceso raíz o sudo al servidor

Instrucciones

  1. Descarga y ejecuta la secuencia de comandos de instalación de Outline.

  2. Sigue las indicaciones que aparecen en pantalla.
