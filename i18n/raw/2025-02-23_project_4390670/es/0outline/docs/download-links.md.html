Si el cliente de Outline se instala tras descargar e instalar el archivo APK de una fuente desconocida, la aplicación no recibirá actualizaciones automáticas, por lo que tendrá que actualizarse manualmente para incluir las funciones más recientes. Para instalar la aplicación, es posible que tengas que cambiar el ajuste para permitir la instalación de aplicaciones desconocidas. Después de instalar Outline, recomendamos restaurar inmediatamente el ajuste para inhabilitar la instalación de aplicaciones desconocidas. Este ajuste evita instalar aplicaciones potencialmente dañinas o maliciosas. ↩
