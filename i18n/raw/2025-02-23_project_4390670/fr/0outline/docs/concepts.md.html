Avant d'utiliser Outline, il peut être utile de revoir ses concepts fondamentaux.

Fonctionnement d'Outline

  1. Outline Manager : vous créez un serveur Outline, sur un fournisseur de services cloud ou sur votre propre matériel, puis vous générez des clés d'accès.
  2. Distribution des clés : vous partagez ces clés avec des proches.
  3. Tunnellisation du trafic : les utilisateurs installent le client Outline et utilisent les clés pour créer un tunnel sécurisé et ainsi contourner la censure.

Clés d'accès

Les clés d'accès sont des identifiants qui permettent aux utilisateurs de se connecter à votre serveur VPN Outline. Elles contiennent les informations nécessaires pour permettre au client Outline d'établir une connexion sécurisée. Il existe deux types de clés d'accès : statique et dynamique.

Clés d'accès statiques

Avec une clé d'accès statique, les informations d'accès au serveur sont encodées directement dans la clé d'accès. Elles ne peuvent ainsi pas être modifiées. Les utilisateurs importent cette clé dans le client Outline pour se connecter au serveur.

  • Exemple : ss://Y2hhY2hhMjAtaWV0Zi1wb2x5MTMwNTo1UkVmeFRqbHR6Mkw@outline-server.example.com:17178/?outline=1

Clés d'accès dynamiques

Une clé d'accès dynamique encode l'emplacement des informations d'accès au serveur. Les accès utilisateur n'ont donc plus besoin d'être associés aux détails d'un serveur spécifique. Vous pouvez ainsi changer, entre autres choses, l'adresse IP, le port ou les paramètres de sécurité de votre serveur sans avoir besoin de mettre à jour les clés d'accès que vous avez déjà distribuées.
