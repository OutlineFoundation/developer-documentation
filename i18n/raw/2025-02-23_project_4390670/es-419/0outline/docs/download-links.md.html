Outline Manager y el cliente de Outline se pueden descargar desde los siguientes vínculos. Obtén más información para configurar Outline en https://getoutline.org/.

Importante: Los siguientes vínculos son las únicas fuentes oficiales para descargar las apps de Outline Manager y el cliente de Outline. No distribuimos directamente estos archivos a través de redes sociales ni apps de mensajería (como Telegram y WhatsApp).

  • Linux: https://s3.amazonaws.com/outline-releases/manager/linux/stable/Outline-Manager.AppImage

  1. Si el cliente de Outline se instala a través de side-loading (es decir, descargando directamente el archivo APK e instalándolo), la app no recibirá actualizaciones automáticas y deberá actualizarse de forma manual para aprovechar las funciones más recientes. Para instalar la app, es posible que debas cambiar tus parámetros de configuración para permitir la instalación de apps desconocidas. Después de instalar Outline, se recomienda cambiar inmediatamente el parámetro de configuración para que se inhabilite la instalación de esas apps. Así, evitarás que se instalen apps potencialmente dañinas o maliciosas de fuentes no seguras. ↩
