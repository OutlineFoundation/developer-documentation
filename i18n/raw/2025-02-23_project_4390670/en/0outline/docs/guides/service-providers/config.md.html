To ensure compatibility across different Outline client versions, use the first-supported option in your configuration. This is particularly important as new strategies and features are added to Outline, as not all users may have updated to the latest client software. By using first-supported, you can provide a single configuration that works seamlessly across various platforms and client versions, ensuring backward compatibility and a consistent user experience.
