---

Это руководство поможет вам настроить сервер Outline для защищенного и неограниченного доступа к интернету.

1. Скачайте Менеджер Outline для Linux, macOS или Windows по ссылке <https://getoutline.org>.

2. В Менеджере Outline нажмите кнопку с плюсом (+), чтобы добавить новый сервер.

3. Выберите поставщика облачных сервисов, например DigitalOcean, Amazon, Google Cloud, или нажмите Set up Outline anywhere (Настроить Outline где угодно), если у вас есть сервер.

4. Следуйте инструкциям на экране для выбранного поставщика или самостоятельной настройки.

5. Менеджер Outline поможет вам создать сервер.

## Дальнейшие действия {#next_steps}

Теперь, когда сервер Outline настроен, вы можете предоставить доступ пользователям.
