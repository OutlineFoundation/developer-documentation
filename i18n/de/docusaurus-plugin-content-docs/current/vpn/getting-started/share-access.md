Nachdem Sie Ihren Outline-Server erfolgreich installiert haben, werden Sie in Outline-Manager auf die Detailseite weitergeleitet. Diese Seite ist Ihr zentraler Hub für die Erstellung und Verwaltung Ihres Servers mit [Zugriffschlüsseln](../../concepts#access-keys).

## Zugriffsschlüssel erstellen und weitergeben {#create_and_share_access_keys}

- **Neuen Zugriffsschlüssel erstellen:** Klicken Sie auf das kleine **+**-Symbol unten auf der Seite, um neue Schlüssel zu erstellen.

- **Neuen Zugriffsschlüssel freigeben:** Rechts neben jedem Schlüssel befindet sich ein Freigabesymbol, auf das Sie klicken können. Daraufhin wird eine Einladung erstellt. Kopieren Sie diese Einladung und geben Sie sie über einen sicheren Kommunikationsweg an den betreffenden Nutzer weiter.

### Prozess zum Einladen von Nutzern {#user_invitation_process}

Nachdem der Nutzer die Einladung erhalten hat, muss er die folgenden Schritte ausführen, um eine Verbindung zu Ihrem Outline-Server herzustellen:

1. Der Nutzer klickt auf seinem Gerät auf den erhaltenen Installationslink.

2. Wenn die Outline-App nicht bereits installiert ist, wird der Nutzer aufgefordert, sie herunterzuladen und zu installieren.

3. Die Outline-App fordert den Nutzer automatisch auf, Ihren Server über den in den Einladungslink eingebetteten eindeutigen Zugriffsschlüssel hinzuzufügen.

4. Der Nutzer kann jetzt die Schaltflächen **Verbinden** und **Trennen** in der Outline-App verwenden, um seinen Zugang zum Internet über Ihren Outline-Server zu steuern.

## Zugriff auf Ihren Server aufheben {#revoke_access_to_your_server}

Die Verwaltung des Nutzerzugriffs ist äußerst wichtig. Outline-Manager bietet eine unkomplizierte Methode, um den Zugriff bei Bedarf aufzuheben.

- **Zugriffsschlüssel suchen:** Gehen Sie in Outline-Manager auf die Detailseite des Servers und suchen Sie den Zugriffsschlüssel des Nutzers, dessen Zugriff Sie aufheben möchten.

- **Zugriffsschlüssel löschen:** Rechts neben dem Schlüssel befindet sich ein Menü-Symbol. Klicken Sie darauf und wählen Sie **Entfernen**.
