---
title: "Deploy Using the Outline Manager"
sidebar_label: "Using Outline Manager"
---

این راهنما شما را در فرایند راه‌اندازی کردن «سرور Outline» همراهی می‌کند تا
دسترسی به اینترنت ایمن و بدون محدودیت را ارائه کند.

1. «مدیر Outline» را برای Linux،‏ macOS، یا Windows از
<https://getoutline.org> بارگیری کنید.

2. در «مدیر Outline»، روی دکمه «+» کلیک کنید تا سرور جدیدی را اضافه کنید.

3. ارائه‌دهنده فضای ابری ترجیحی‌تان را انتخاب کنید (برای نمونه، ‫DigitalOcean،‏ Amazon،‏ Google Cloud) یا اگر سرور خودتان را دارید، «Outline را در هر جایی راه‌اندازی کنید».

4. از دستورالعمل‌های روی صفحه که مخصوص ارائه‌دهنده منتخب شما است پیروی کنید یا
خودتان راه‌اندازی کنید.

5. «مدیر Outline» شما را در ساختن سرور راهنمایی خواهد کرد.

## مراحل بعدی

اکنون که سرور Outline شما راه‌اندازی شده است، برای هم‌رسانی کردن دسترسی به
کاربرانتان آماده‌اید.
