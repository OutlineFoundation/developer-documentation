این راهنما شما را در فرایند راه‌اندازی کردن Go environment و
استفاده کردن از Outline SDK به زبان Go code کمک می‌کند.

ما برنامه نمونه‌ای به‌نام `splitfetch` خواهیم ساخت که
ویژگی «کیت توسعه نرم‌افزار» را نمایش می‌دهد. این برنامه صفحه وب را واکشی می‌کند ولی به‌جای اینکه
درخواست را در یک بسته شبکه‌ای ارسال کند، **از Outline SDK استفاده می‌کند تا
جاری‌سازی TCP اولیه را به دو بسته جداگانه تقسیم کند**. این کار ممکن است کمک کند تا برخی‌از فرم‌های
مداخله شبکه را دور بزنید.

می‌توانید برنامه را در **Linux،‏ Mac، و Windows** اجرا کنید.
برای ادغام کردن با برنامه‌های تلفن همراه، [افزودن Outline SDK به برنامه تلفن همراهتان](mobile-app-integration) را ببینید.
