Outline 서버 설치를 완료하면 Outline Manager 내의 서버 세부정보 페이지로 이동하게 됩니다. 이 페이지는 [액세스 키](../../concepts#access-keys)를 사용하여 서버에 대한 액세스 권한을 생성하고 관리하는 중앙 허브입니다.

## 액세스 키 생성 및 공유하기 {#create_and_share_access_keys}

- **새 키 만들기:** 페이지 하단의 작은 **+** 아이콘을 클릭하여 새 키를 만듭니다.

- **새 키 공유하기:** 각 키의 오른쪽에 있는 공유 아이콘을 클릭하면 초대 링크를 확인할 수 있습니다. 이 초대 링크를 복사하여 원하는 메시지 플랫폼을 통해 의도한 사용자에게 전달합니다.

### 사용자 초대 프로세스 {#user_invitation_process}

초대를 받은 사용자는 다음 단계를 따라 Outline 서버에 연결합니다.

1. 사용자는 제공된 설치 링크를 자신의 기기에서 클릭합니다.

2. Outline 앱이 아직 설치되어 있지 않은 경우 앱을 다운로드하고 설치하라는 메시지가 사용자에게 표시됩니다.

3. Outline 앱은 초대 링크에 포함된 고유 액세스 키를 통해 사용자에게 서버를 추가하라는 메시지를 자동으로 표시합니다.

4. 사용자는 이제 Outline 앱 내의 **연결** 및 **연결 해제** 버튼을 사용하여 Outline 서버를 통해 인터넷 액세스를 관리할 수 있습니다.

## 서버에 대한 액세스 권한 취소하기 {#revoke_access_to_your_server}

사용자 액세스 관리는 매우 중요합니다. Outline Manager를 사용하면 필요 시 간편하게 액세스 권한을 취소할 수 있습니다.

- **키 찾기:** Outline Manager에서 서버 세부정보 페이지로 이동한 다음 액세스 권한을 취소하려는 사용자의 액세스 키를 확인합니다.

- **키 삭제하기:** 해당 키 오른쪽에는 메뉴 아이콘이 있습니다. 메뉴 아이콘을 클릭하고 **삭제**를 선택합니다.
