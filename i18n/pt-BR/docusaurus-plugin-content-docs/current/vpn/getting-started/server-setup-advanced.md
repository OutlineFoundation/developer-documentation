Este guia orienta todo o processo de configuração do servidor do Outline para
você ter um acesso seguro e irrestrito à internet.

## Pré-requisitos

- Um servidor (físico ou virtual) com sistema operacional aceito (Ubuntu
20.04 LTS ou Debian 10)

- Acesso root ou sudo ao servidor

## Instruções

1. 

Faça o download e execute o script de instalação do Outline.

2. 

Siga as instruções na tela.
