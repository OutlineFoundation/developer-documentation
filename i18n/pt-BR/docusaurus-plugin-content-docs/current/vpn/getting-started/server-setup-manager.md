Este guia orienta todo o processo de configuração do servidor do Outline para
você ter acesso seguro e irrestrito à internet.

1. Faça o download do Outline Manager para Linux, macOS ou Windows em
<https://getoutline.org>.

2. No Outline Manager, clique no botão "+" para adicionar um novo servidor.

3. Escolha um provedor de nuvem (por ex.: DigitalOcean, Amazon, Google
Cloud) ou clique em "Configure o Outline em qualquer lugar" se você tiver seu próprio servidor.

4. Siga as instruções na tela específicas para o provedor escolhido ou
para o seu próprio servidor.

5. O Outline Manager vai guiar você durante a criação do servidor.
