使用安装脚本进行部署

本指南详细介绍了设置 Outline 服务器以提供安全无限制的互联网访问服务的完整流程。

前提条件​

  • 运行受支持的操作系统（Ubuntu 20.04 LTS 或 Debian 10）的服务器（物理服务器或虚拟服务器）

  • 对服务器的根访问权限或 sudo 访问权限

操作说明​

  1. 下载并运行 Outline 安装脚本。
sudo bash -c "$(wget -qO- https://raw.githubusercontent.com/OutlineFoundation/outline-apps/master/server_manager/install_scripts/install_server.sh)"
  1. 按照屏幕上的提示操作。
