вҖ«Outline ШҜЩҲ ЩҶЩҲШ№ Ъ©Щ„ЫҢШҜ ШҜШіШӘШұШіЫҢ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ: Ш«Ш§ШЁШӘ ЩҲ ЩҫЩҲЫҢШ§. ШҜШұ Ъ©ШҜШЁЩҶШҜЫҢ Ъ©Щ„ЫҢШҜЩҮШ§ЫҢ Ш«Ш§ШЁШӘ ЩҮЩ…ЩҮ Ш§Ш·Щ„Ш§Ш№Ш§ШӘЩҗ Ш§ШӘШөШ§Щ„ ШҜШұЩҲЩҶ Ш®ЩҲШҜ Ъ©ШҜ Ш§ШіШӘШҢ ШҜШұШӯШ§Щ„ЫҢвҖҢЪ©ЩҮ ШҜШұ Ъ©ШҜШЁЩҶШҜЫҢ Ъ©Щ„ЫҢШҜЩҮШ§ЫҢ ЩҫЩҲЫҢШ§ Щ…Ъ©Ш§ЩҶ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Ш§ШӘШөШ§Щ„ ШЁЩҮ ШҙЩ…Ш§ Ш§Ш¬Ш§ШІЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ ШӘШ§ ШўЩҶ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШұШ§ ШҜШұ Ш¬Ш§ЫҢЫҢ ШҜЩҲШұШӘШұ Ш°Ш®ЫҢШұЩҮ Ъ©ЩҶЫҢШҜ ЩҲ ШҜШұШөЩҲШұШӘ ЩҶЫҢШ§ШІШҢ ШўЩҶ ШұШ§ ШӘШәЫҢЫҢШұ ШҜЩҮЫҢШҜ. Ш§ЫҢЩҶ ЫҢШ№ЩҶЫҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ ШіШұЩҲШұ Ш®ЩҲШҜ ШұШ§ ШЁЩҮвҖҢШұЩҲШІ Ъ©ЩҶЫҢШҜШҢ ШЁШҜЩҲЩҶ Ш§ЫҢЩҶЪ©ЩҮ Щ…Ш¬ШЁЩҲШұ ШЁШ§ШҙЫҢШҜ Ъ©Щ„ЫҢШҜЩҮШ§ЫҢ Ш¬ШҜЫҢШҜЫҢ ШӘЩҲЩ„ЫҢШҜ Ъ©ЩҶЫҢШҜ ЩҲ Щ…ЫҢШ§ЩҶ Ъ©Ш§ШұШЁШұШ§ЩҶШӘШ§ЩҶ ШӘЩҲШІЫҢШ№ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ ШіЩҶШҜ ШӘЩҲШ¶ЫҢШӯ Щ…ЫҢвҖҢШҜЩҮШҜ ШЁШұШ§ЫҢ ШӘШ№Ш§Щ…Щ„ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұШӘШұ ЩҲ Ъ©Ш§ШұШўЩ…ШҜШӘШұ ШЁШ§ ШіШұЩҲШұ Outline Ш®ЩҲШҜШӘШ§ЩҶШҢ ЪҶШ·ЩҲШұ Ш§ШІ Ъ©Щ„ЫҢШҜЩҮШ§ЫҢ ШҜШіШӘШұШіЫҢ ЩҫЩҲЫҢШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.

ШЁШұШ§ЫҢ Щ…Ш№ЫҢЩҶ Ъ©ШұШҜЩҶ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШҜШіШӘШұШіЫҢ Ъ©ЩҮ ШӘЩҲШіШ· Ъ©Щ„ЫҢШҜЩҮШ§ЫҢ ШҜШіШӘШұШіЫҢ ЩҫЩҲЫҢШ§ЫҢ ШҙЩ…Ш§ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲШҜШҢ ШіЩҮ ЩӮШ§Щ„ШЁ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ:

Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ЩҫЫҢЩҲЩҶШҜ ss://вҖӢ

Ъ©Ш§ШұШ®ЩҲШ§ЩҮ Outline ЩҶШіШ®ЩҮ Ыұ.Ыё.Ыұ ШЁЩҮ ШЁШ§Щ„Ш§.

Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Щ…ШіШӘЩӮЫҢЩ…Ш§ЩӢ Ш§ШІ ЩҫЫҢЩҲЩҶШҜ ss:// Ъ©ЩҶЩҲЩҶЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ. Ш§ЫҢЩҶ ШұЩҲШҙ ЩҲЩӮШӘЫҢ Ш§ЫҢШҜЩҮвҖҢШўЩ„ Ш§ШіШӘ Ъ©ЩҮ ШЁЩҮ ШӘШәЫҢЫҢШұ ШҜШ§ШҜЩҶ Щ…ШұШӘШЁ ШіШұЩҲШұШҢ ШҜШұЪҜШ§ЩҮШҢ ЫҢШ§ ШұЩҲШҙ ШұЩ…ШІЪҜШ°Ш§ШұЫҢвҖҢШӘШ§ЩҶ ЩҶЫҢШ§ШІЫҢ ЩҶШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜ ЩҲЩ„ЫҢ ЩҮЩҶЩҲШІ ШЁШ®ЩҲШ§ЩҮЫҢШҜ ШЁШұШ§ЫҢ ШЁЩҮвҖҢШұЩҲШІ Ъ©ШұШҜЩҶ ЩҶШҙШ§ЩҶЫҢ ШіШұЩҲШұ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜ.

ss://Y2hhY2hhMjAtaWV0Zi1wb2x5MTMwNTpleGFtcGxl@outline-server.example.com:8388/?outline=1

Ш§ЫҢЩҶ ШұЩҲШҙ ШЁШұШ§ЫҢ Щ…ШҜЫҢШұЫҢШӘ Ъ©ШұШҜЩҶ ЩҮЩ…ЩҮ ЩҶЩ…ЩҲШҜЩҮШ§ЫҢ Ш§ШӘШөШ§Щ„ Ъ©Ш§ШұШЁШұШ§ЩҶ Outline Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ ШЁЫҢШҙШӘШұЫҢ ШұШ§ Ш§ШұШ§ШҰЩҮ Щ…ЫҢвҖҢШҜЩҮШҜ. ШЁШ§ Ш§ЫҢЩҶ ШұЩҲШҙ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШіШұЩҲШұШҢ ШҜШұЪҜШ§ЩҮШҢ ЪҜШ°ШұЩҲШ§ЪҳЩҮШҢ ЩҲ ШұЩҲШҙ ШұЩ…ШІЪҜШ°Ш§ШұЫҢ ШұШ§ ШЁЩҮвҖҢШұЩҲШІ Ъ©ЩҶЫҢШҜ.

ЩҶЩ…ЩҲЩҶЩҮ:

  • вҖ«server_port: ШҙЩ…Ш§ШұЩҮ ШҜШұЪҜШ§ЩҮЫҢ Ъ©ЩҮ ШіШұЩҲШұ ЩҲЫҢвҖҢЩҫЫҢвҖҢШ§ЩҶ ШҙЩ…Ш§ ШҜШұ ШўЩҶ Ш§Ш¬ШұШ§ Щ…ЫҢвҖҢШҙЩҲШҜ.

  • ЪҜШ°ШұЩҲШ§ЪҳЩҮ: ЪҜШ°ШұЩҲШ§ЪҳЩҮвҖҢШ§ЫҢ Ъ©ЩҮ ШЁШұШ§ЫҢ Ш§ШӘШөШ§Щ„ ШЁЩҮ ЩҲЫҢвҖҢЩҫЫҢвҖҢШ§ЩҶ Щ„Ш§ШІЩ… Ш§ШіШӘ.

  • ШұЩҲШҙ: ШұЩҲШҙ ШұЩ…ШІЪҜШ°Ш§ШұЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮвҖҢШҙШҜЩҮ ШӘЩҲШіШ· ЩҲЫҢвҖҢЩҫЫҢвҖҢШ§ЩҶ. ШЁЩҮ Shadowsocks Ъ©ЩҮ ШұЩ…ШІЩҮШ§ЫҢ AEAD ШұШ§ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ Ш§ШұШ¬Ш§Ш№ ШҜЩҮЫҢШҜ.

Ш§ШіШӘЩҒШ§ШҜЩҮ Ш§ШІ ШҙЫҢШЎ YAMLвҖӢ

Ъ©Ш§ШұШ®ЩҲШ§ЩҮ Outline ЩҶШіШ®ЩҮ Ыұ.ЫұЫө.Ы° ШЁЩҮ ШЁШ§Щ„Ш§.

Ш§ЫҢЩҶ ШұЩҲШҙ ШҙШЁЫҢЩҮ ШұЩҲШҙ JSON ЩӮШЁЩ„ЫҢ Ш§ШіШӘ Ш§Щ…Ш§ ШЁШ§ ЩӮШҜШұШӘ ЩҶЩҒЩҲШ° ЩӮШ§Щ„ШЁ ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ ЩҫЫҢШҙШұЩҒШӘЩҮ OutlineШҢ Ш§ЩҶШ№Ш·Ш§ЩҒвҖҢЩҫШ°ЫҢШұЫҢ ШЁЫҢШҙШӘШұЫҢ ШЁЩҮ ШўЩҶ Ш§Ш¶Ш§ЩҒЩҮ ШҙШҜЩҮ Ш§ШіШӘ. ШЁШ§ Ш§ЫҢЩҶ ШұЩҲШҙ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШіШұЩҲШұШҢ ШҜШұЪҜШ§ЩҮШҢ ЪҜШ°ШұЩҲШ§ЪҳЩҮШҢ ШұЩҲШҙ ШұЩ…ШІЪҜШ°Ш§ШұЫҢШҢ ЩҲ Щ…ЩҲШ§ШұШҜ ШЁШіЫҢШ§Шұ ШЁЫҢШҙШӘШұЫҢ ШұШ§ ШЁЩҮвҖҢШұЩҲШІ Ъ©ЩҶЫҢШҜ.

    secret: example

  • ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„: ЩҫШұЩҲШӘЪ©Щ„вҖҢЩҮШ§ЫҢ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ЫҢ ШұШ§ Ъ©ЩҮ Ш§ШіШӘЩҒШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ ШӘШ№ШұЫҢЩҒ Щ…ЫҢвҖҢЪ©ЩҶШҜ (ШҜШұ Ш§ЫҢЩҶ ШӯШ§Щ„ШӘШҢ TCP ЩҲ UDP).

  • вҖ«tcp/udp: ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ ШұШ§ ШЁШұШ§ЫҢ ЩҮШұ ЩҫШұЩҲШӘЪ©Щ„ Щ…ШҙШ®Шө Щ…ЫҢвҖҢЪ©ЩҶШҜ.

    • вҖ«$ЩҶЩҲШ№: ЩҶЩҲШ№ ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ ШұШ§ Щ…ШҙШ®Шө Щ…ЫҢвҖҢЪ©ЩҶШҜШҢ ШҜШұ Ш§ЫҢЩҶШ¬Ш§ Shadowsocks Ш§ШіШӘ.

    • ЩҶЩӮШ·ЩҮ ЩҫШ§ЫҢШ§ЩҶЫҢ: ШҜШ§Щ…ЩҶЩҮ ЫҢШ§ ЩҶШҙШ§ЩҶЫҢ IP ЩҲ ШҜШұЪҜШ§ЩҮ Щ…ШұШЁЩҲШ· ШЁЩҮ ШіШұЩҲШұ ЩҲЫҢвҖҢЩҫЫҢвҖҢШ§ЩҶ ШҙЩ…Ш§ Ш§ШіШӘ.

    • ШұШ§ШІ: ЪҜШ°ШұЩҲШ§ЪҳЩҮвҖҢШ§ЫҢ Ъ©ЩҮ ШЁШұШ§ЫҢ Ш§ШӘШөШ§Щ„ ШЁЩҮ ЩҲЫҢвҖҢЩҫЫҢвҖҢШ§ЩҶ Щ„Ш§ШІЩ… Ш§ШіШӘ.

    • ШұЩ…ШІ: ШұЩҲШҙ ШұЩ…ШІЪҜШ°Ш§ШұЫҢ Ш§ШіШӘЩҒШ§ШҜЩҮвҖҢШҙШҜЩҮ ШӘЩҲШіШ· ЩҲЫҢвҖҢЩҫЫҢвҖҢШ§ЩҶ. ШЁЩҮ Shadowsocks Ъ©ЩҮ ШұЩ…ШІЩҮШ§ЫҢ AEAD ШұШ§ ЩҫШҙШӘЫҢШЁШ§ЩҶЫҢ Щ…ЫҢвҖҢЪ©ЩҶШҜ Ш§ШұШ¬Ш§Ш№ ШҜЩҮЫҢШҜ.

ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ Ъ©Щ„ЫҢШҜ ШҜШіШӘШұШіЫҢ ШұШ§ ШЁШЁЫҢЩҶЫҢШҜ ШӘШ§ ШҜШұШЁШ§ШұЩҮ ШұШ§ЩҮвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ШҜШіШӘШұШіЫҢ ШЁЩҮ ШіШұЩҲШұ Outline Ш®ЩҲШҜ ШұШ§ ЩҫЫҢЪ©ШұШЁЩҶШҜЫҢ Ъ©ЩҶЫҢШҜШҢ Ш§ШІШ¬Щ…Щ„ЩҮ ШӯЩ…Щ„вҖҢЩҲЩҶЩӮЩ„ШҢ ЩҶЩӮШ·ЩҮвҖҢЩҮШ§ЫҢ ЩҫШ§ЫҢШ§ЩҶЫҢШҢ ШҙЩ…Ш§ШұЩҮвҖҢЪҜЫҢШұЩҮШ§ШҢ ЩҲ ШҙЩҶЩҲЩҶШҜЪҜШ§ЩҶ ШЁШіШӘЩҮ Ш§Ш·Щ„Ш§Ш№Ш§ШӘЫҢ Ъ©ШіШЁ Ъ©ЩҶЫҢШҜ.

Ш§ШіШӘШ®ШұШ§Ш¬ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШҜШіШӘШұШіЫҢ Ш§ШІ В«Ъ©Щ„ЫҢШҜ Ш«Ш§ШЁШӘВ»вҖӢ

Ш§ЪҜШұ ЩҒШ№Щ„Ш§ЩӢ Ъ©Щ„ЫҢШҜ ШҜШіШӘШұШіЫҢ Ш«Ш§ШЁШӘ ШҜШ§ШұЫҢШҜШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ ШўЩҶ ШұШ§ Ш§ШіШӘШ®ШұШ§Ш¬ Ъ©ЩҶЫҢШҜ ШӘШ§ Ъ©Щ„ЫҢШҜ ШҜШіШӘШұШіЫҢ ЩҫЩҲЫҢШ§ЫҢЫҢ ШЁШұЩҫШ§ЫҢЩҮ JSON ЫҢШ§ YAML ШЁШіШ§ШІЫҢШҜ. Ъ©Щ„ЫҢШҜЩҮШ§ЫҢ ШҜШіШӘШұШіЫҢ Ш«Ш§ШЁШӘ Ш§ШІ Ш§Щ„ЪҜЩҲЫҢ ШІЫҢШұ ЩҫЫҢШұЩҲЫҢ Щ…ЫҢвҖҢЪ©ЩҶЩҶШҜ:

SS-URI = "ss://" userinfo "@" hostname ":" port [ "/" ] [ "#" tag ]
userinfo = websafe-base64-encode-utf8(method  ":" password)
           method ":" password
