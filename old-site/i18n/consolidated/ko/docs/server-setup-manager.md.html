이 가이드에서는 안전하고 제한 없는 인터넷 액세스를 제공하기 위해 Outline 서버를 설정하는 과정을 안내합니다.

단계별 안내

  1. https://getoutline.org에서 Linux, macOS, Windows용 Outline Manager를 다운로드합니다.

  2. Outline Manager에서 '+' 버튼을 클릭하여 새 서버를 추가합니다.

  3. 기본 클라우드 제공업체(예: DigitalOcean, Amazon, Google Cloud)를 선택하거나 자체 서버가 있는 경우 'Set up Outline anywhere'(어디서나 Outline 설정)를 선택합니다.

  4. 선택한 제공업체와 관련하여 화면에 표시된 안내를 따르거나 직접 설정합니다.

  5. Outline Manager에서는 서버를 만들고 Outline을 설치하고 액세스 키를 생성하는 방법을 안내합니다.
