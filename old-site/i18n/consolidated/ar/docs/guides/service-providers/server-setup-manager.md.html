سنتعرف في هذا الدليل على عملية إعداد خادم Outline لتوفير إمكانية وصول إلى الإنترنت آمنة وغير مقيّدة.

  1. نزِّل تطبيق "مدير Outline" لأنظمة التشغيل Linux أو macOS أو Windows من الرابط https://getoutline.org.

  2. في تطبيق "مدير Outline"، انقر على زر + لإضافة خادم جديد.

  3. اختَر مقدّم خدمات السحابة الإلكترونية المفضّل لديك (مثل DigitalOcean أو Amazon أو Google Cloud) أو انقر على "إعداد Outline في أي مكان" إذا كان لديك خادمك الخاص.

  4. اتّبِع التعليمات الظاهرة على الشاشة والخاصة بمقدّم خدمات السحابة الإلكترونية الذي اخترته أو بعملية الإعداد اليدوي.

  5. سيرشدك تطبيق "مدير Outline" عبر خطوات إنشاء خادم.
