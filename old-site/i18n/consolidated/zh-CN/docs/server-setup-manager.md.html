本指南详细介绍了设置 Outline 服务器以提供安全无限制的互联网访问服务的完整流程。

分步说明

  1. 从 https://getoutline.org 下载适用于 Linux、macOS 或 Windows 的 Outline 管理器。

  2. 在 Outline 管理器中点击“+”按钮，以添加新服务器。

  3. 选择首选云提供商（例如 DigitalOcean、Amazon、Google Cloud）；如果您有自己的服务器，则选择“随时随地安装 Outline”。

  4. 按照屏幕上针对所选提供商的说明或自助安装说明操作。

  5. Outline 管理器将会全程引导您创建服务器、安装 Outline 并生成访问密钥。
