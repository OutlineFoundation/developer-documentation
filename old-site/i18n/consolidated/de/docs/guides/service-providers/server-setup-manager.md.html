In diesem Leitfaden werden Sie Schritt für Schritt durch die Einrichtung eines Outline-Servers geführt, um einen sicheren und uneingeschränkten Internetzugriff bereitzustellen.

  1. Laden Sie unter https://getoutline.org den Outline-Manager für Linux, macOS oder Windows herunter.

  2. Klicken Sie im Outline-Manager auf die Schaltfläche „+“. Damit wird ein neuer Server hinzugefügt.

  3. Wählen Sie Ihren bevorzugten Cloud-Anbieter aus (z. B. DigitalOcean, Amazon, Google Cloud). Wenn Sie Ihren eigenen Server haben, wählen Sie „Outline überall einrichten“ aus.

  4. Nutzen Sie einen Cloud-Anbieter, folgen Sie der entsprechenden Anleitung auf dem Bildschirm. Möchten Sie die Einrichtung selbst vornehmen, folgen Sie der Anleitung zur selbstständigen Einrichtung.

  5. Der Outline-Manager zeigt Ihnen Schritt für Schritt, wie Sie einen Server erstellen.

Nächste Schritte

Jetzt ist Ihr Outline-Server eingerichtet und Sie können den Zugriff für Ihre Nutzer freigeben.
