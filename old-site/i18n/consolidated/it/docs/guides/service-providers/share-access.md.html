Dopo aver completato l'installazione del server Outline, visualizzerai la pagina dei dettagli del server in Outline Manager. Questa pagina funge da hub centrale per generare le chiavi di accesso e gestire l'accesso al server.

Crea e condividi le chiavi di accesso

  • Per creare una nuova chiave: fai clic sulla piccola icona + in fondo alla pagina per creare nuove chiavi.
  • Per condividere la nuova chiave: fai clic sull'icona di condivisione a destra di ogni chiave per generare un invito. Copia l'invito e invialo all'utente desiderato attraverso la piattaforma di messaggistica che preferisci.
Suggerimento: usa una piattaforma di messaggistica affidabile per condividere i link di invito, ad esempio email criptate o un'applicazione di chat sicura. Evita di utilizzare canali non protetti, per non rischiare di compromettere la chiave di accesso.

Procedura di invito degli utenti

Ecco la procedura che segue l'utente per connettersi al tuo server Outline dopo aver ricevuto l'invito.

  1. L'utente fa clic sul link di installazione fornito sul suo dispositivo.
  2. Se l'app Outline non è già installata, l'utente visualizza la richiesta di scaricarla e installarla.
  3. L'app Outline chiede automaticamente all'utente di aggiungere il tuo server usando la chiave di accesso univoca incorporata nel link di invito.
  4. Ora l'utente può usare i pulsanti Connetti e Disconnetti nell'app Outline per controllare il suo accesso a internet tramite il tuo server Outline.

Revoca l'accesso al tuo server

Gestire l'accesso degli utenti è fondamentale. Outline Manager offre un metodo semplice per revocare l'accesso quando è necessario.

  • Trova la chiave: vai alla pagina dei dettagli del server in Outline Manager e individua la chiave di accesso corrispondente all'utente di cui vuoi revocare l'accesso.
  • Elimina la chiave: a destra della chiave troverai un'icona di menu. Fai clic sull'icona di menu e seleziona Rimuovi.
