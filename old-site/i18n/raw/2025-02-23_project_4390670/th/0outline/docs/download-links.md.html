Outline Manager และไคลเอ็นต์ Outline สามารถดาวน์โหลดได้จากลิงก์ด้านล่างนี้ โปรดดูข้อมูลเพิ่มเติมเกี่ยวกับการตั้งค่า Outline ที่ https://getoutline.org/

สำคัญ: ลิงก์ต่อไปนี้เป็นแหล่งที่มาอย่างเป็นทางการแหล่งเดียวในการดาวน์โหลดแอปพลิเคชัน Outline Manager และไคลเอ็นต์ Outline เท่านั้น เราไม่ได้แจกจ่ายไฟล์เหล่านี้โดยตรงผ่านโซเชียลมีเดียหรือแอปรับส่งข้อความต่างๆ ซึ่งรวมถึง Telegram และ WhatsApp
