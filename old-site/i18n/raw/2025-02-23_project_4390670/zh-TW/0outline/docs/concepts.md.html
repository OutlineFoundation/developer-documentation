使用 Outline 前，建議瞭解一些核心概念。

Outline 的運作方式

  1. Outline Manager：您可以在雲端服務供應商或自己的硬體上建立 Outline 伺服器，然後產生存取金鑰。
  2. 發布金鑰：與親友分享金鑰。
  3. 通道流量：使用者安裝 Outline 用戶端，並利用金鑰建立安全通道，從而避開審查。

存取金鑰

存取金鑰是使用者連線至 Outline VPN 伺服器的憑證，內含 Outline 用戶端建立安全連線所需的資訊。存取金鑰分為靜態與動態兩種。

靜態存取金鑰

靜態存取金鑰會將伺服器存取資訊直接編碼在金鑰內，防止資訊遭到修改。使用者只需將金鑰匯入 Outline 用戶端，即可連線至伺服器。

  • 示例： ss://Y2hhY2hhMjAtaWV0Zi1wb2x5MTMwNTo1UkVmeFRqbHR6Mkw@outline-server.example.com:17178/?outline=1

動態存取金鑰

動態存取金鑰編碼的是伺服器存取資訊的位置，所以使用者不必依賴特定伺服器詳細資料來連線。也就是說，即使您更改伺服器的 IP 位址、通訊埠、安全設定等，也無需更新已發布的存取金鑰。

如需更多資訊，請參閱「動態存取金鑰」一文。
