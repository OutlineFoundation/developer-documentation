Antes de usar Outline, es buena idea que revises los conceptos fundamentales.

Cómo funciona Outline

  1. Outline Manager: Puedes crear un servidor de Outline en un proveedor de servicios en la nube o en tu propio hardware y, luego, generar claves de acceso.
  2. Distribuye claves: Puedes compartir estas claves con tus amistades y familiares.
  3. Tráfico de túnel: Los usuarios instalan el cliente de Outline y usan las claves para establecer un túnel seguro y, así, evitar la censura.

Claves de acceso

Las claves de acceso son las credenciales que permiten a los usuarios conectarse al servidor de VPN de Outline. Contienen la información necesaria para que el cliente de Outline establezca una conexión segura. Existen dos tipos de claves de acceso: estáticas y dinámicas.

Claves de acceso estáticas

En una clave de acceso estática, la información de acceso al servidor se codifica directamente en la clave, lo que impide que se modifique la información de acceso. Los usuarios importan esta clave al cliente de Outline para conectarse al servidor.

  • Ejemplo: ss://Y2hhY2hhMjAtaWV0Zi1wb2x5MTMwNTo1UkVmeFRqbHR6Mkw@outline-server.example.com:17178/?outline=1

Claves de acceso dinámicas

Una clave de acceso dinámica codifica la ubicación de la información de acceso al servidor, lo que permite desvincular el acceso de los usuarios de los detalles específicos del servidor. Esto significa que puedes cambiar la dirección IP, el puerto, la configuración de seguridad y otros aspectos del servidor sin tener que actualizar las claves de acceso ya distribuidas.

Consulta la documentación detallada sobre las claves de acceso dinámicas para obtener más información.
