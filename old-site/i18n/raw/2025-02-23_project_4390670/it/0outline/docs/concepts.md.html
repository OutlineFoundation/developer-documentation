Prima di utilizzare Outline, può essere utile ripassare i concetti fondamentali.

Come funziona Outline

  1. Outline Manager: crei un server Outline, su un provider cloud o sul tuo hardware, e poi generi le chiavi di accesso.
  2. Distribuisci le chiavi: condividi queste chiavi con amici e familiari.
  3. Traffico in tunnel: gli utenti installano il client Outline e usano le chiavi per stabilire un tunnel sicuro, aggirando la censura.

Chiavi di accesso

Le chiavi di accesso sono le credenziali che consentono agli utenti di connettersi al server VPN Outline. Contengono le informazioni necessarie affinché il client Outline stabilisca una connessione sicura. Esistono due tipi di chiavi di accesso: statiche e dinamiche.

Chiavi di accesso statiche

In una chiave di accesso statica, le informazioni di accesso al server sono codificate direttamente nella chiave di accesso. Ciò impedisce che le informazioni di accesso vengano modificate. Gli utenti importano questa chiave nel client Outline per connettersi al server.

  • Esempio: ss://Y2hhY2hhMjAtaWV0Zi1wb2x5MTMwNTo1UkVmeFRqbHR6Mkw@outline-server.example.com:17178/?outline=1

Chiavi di accesso dinamiche

Una chiave di accesso dinamica codifica la posizione delle informazioni di accesso al server, consentendoti di scollegare l'accesso utente da dettagli specifici del server. Ciò significa che puoi modificare l'indirizzo IP, la porta, le impostazioni di sicurezza e altri valori del tuo server senza dover aggiornare le chiavi di accesso che hai già distribuito.

Per ulteriori informazioni, consulta la documentazione dettagliata sulle chiavi di accesso dinamiche.
