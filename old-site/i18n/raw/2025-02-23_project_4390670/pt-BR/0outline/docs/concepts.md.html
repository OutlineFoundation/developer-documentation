Antes de usar o Outline, recomendamos rever os conceitos básicos.

Como o Outline funciona

  1. Gerenciador do Outline: você cria um servidor Outline, seja em um provedor de nuvem ou no seu próprio hardware, e então gera chaves de acesso.
  2. Distribuição das chaves: você compartilha essas chaves com amigos e familiares.
  3. Tráfego de túnel: os usuários instalam o app cliente do Outline e criam um túnel seguro com as chaves, contornando a censura.

Chaves de acesso

As chaves de acesso são as credenciais que permitem aos usuários se conectar ao servidor VPN do Outline. Elas contêm as informações necessárias para que o app cliente do Outline estabeleça uma conexão segura. Existem dois tipos de chaves de acesso: estáticas e dinâmicas.

Chaves de acesso estáticas

Em uma chave de acesso estática, as informações de acesso ao servidor são codificadas diretamente na chave de acesso. Isso evita que as informações de acesso sejam modificadas. Os usuários importam essa chave para o app cliente do Outline a fim de se conectar ao servidor.

  • Exemplo: ss://Y2hhY2hhMjAtaWV0Zi1wb2x5MTMwNTo1UkVmeFRqbHR6Mkw@outline-server.example.com:17178/?outline=1

Chaves de acesso dinâmicas

Uma chave de acesso dinâmica codifica a localização das informações de acesso ao servidor, permitindo que você desvincule o acesso do usuário de detalhes específicos do servidor. Isso significa que você pode alterar o endereço IP, a porta, as configurações de segurança e muito mais do seu servidor, sem precisar atualizar as chaves de acesso que já distribuiu.

Confira a documentação detalhada sobre chaves de acesso dinâmicas para mais informações.
