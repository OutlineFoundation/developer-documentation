  1. Se o app cliente do Outline for transferido por sideload com o download e a instalação do arquivo APK, ele não receberá atualizações automáticas. É preciso atualizá-lo manualmente para usar os recursos mais recentes. Para instalar o app, talvez seja necessário mudar as configurações e permitir a instalação de apps desconhecidos. Recomendamos que você volte para a configuração original, que impede a instalação de apps desconhecidos, logo depois de instalar o Outline. Isso evita a instalação de apps possivelmente nocivos ou maliciosos de fontes não seguras. ↩
