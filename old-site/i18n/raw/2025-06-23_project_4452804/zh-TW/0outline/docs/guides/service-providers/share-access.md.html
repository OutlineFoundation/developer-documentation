Outline 伺服器安裝成功後，系統會將您導向 Outline Manager 的伺服器詳細資料頁面。您可以在這個頁面，建立及管理用於存取伺服器的存取金鑰。

建立及共用存取金鑰

  • 建立新金鑰：點選頁面底部的「+」小型圖示，即可建立新金鑰。
  • 共用新金鑰：每組金鑰右側都有共用圖示，點選即可產生邀請連結。請複製這個連結，然後用偏好的通訊方式傳給目標使用者。
提示：分享邀請連結時，請使用安全的通訊方式，例如加密電子郵件或安全的即時通訊應用程式。避免透過不安全的管道傳送，以免存取金鑰外洩。

使用者邀請流程

使用者收到邀請後，會依照以下步驟連線到您的 Outline 伺服器：

  1. 在裝置上點選您提供的安裝連結。
  2. 如果尚未安裝 Outline 應用程式，系統會引導使用者下載並安裝。
  3. Outline 應用程式會自動提示使用者，用邀請連結內的專屬存取金鑰加入您的伺服器。
  4. 接著，使用者就能隨時點選 Outline 應用程式的「連線」或「中斷連線」按鈕，自由控制是否透過您的 Outline 伺服器上網。

撤銷伺服器存取權

管理使用者存取權至關重要。如果需要撤銷某位使用者的權限，可以透過 Outline Manager 輕鬆完成。

  • 找出金鑰：前往 Outline Manager 的伺服器詳細資料頁面，找出該使用者對應的存取金鑰。
  • 刪除金鑰：點選金鑰右側的選單圖示，然後選取「移除」。
