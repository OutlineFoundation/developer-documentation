After successfully installing your Outline server, you'll be directed to the server details page within Outline Manager. This page is your central hub for generating and managing access to your server using access keys.

Create and share access keys

  • Create a new key: Click the small + icon at the bottom of the page to create new keys.
  • Share the new key: To the right of each key, there's a share icon you can click to get an invitation. Copy this invitation and distribute it to the intended user through your preferred messaging platform.
Tip: Use a secure messaging platform such as encrypted email or a secure chat application when sharing invitation links. Avoid using unsecured channels as this could compromise the access key.

User invitation process

Upon receiving the invitation, the user will follow these steps to connect to your Outline server:

  1. The user clicks the provided installation link on their device.
  2. If the Outline app is not already installed, the user will be prompted to download and install it.
  3. The Outline app automatically prompts the user to add your server using the unique access key embedded within the invitation link.
  4. The user can now use the Connect and Disconnect buttons within the Outline app to control their access to the internet through your Outline server.

Revoke access to your server

Managing user access is crucial. Outline Manager provides a straightforward method for revoking access when necessary.

  • Locate the key: Navigate to the server details page in Outline Manager and identify the access key corresponding to the user whose access you want to revoke.
  • Delete the key: To the right of the key, you'll find a menu icon. Click the menu icon and select Remove.
