Before you use Outline, it can be helpful to review the core concepts.

How Outline Works

  • Blocked access: Users face restrictions in accessing the open internet.
  • Outline Manager: You create a server on the cloud and generate access keys.
  • Distribute keys: Share these keys with friends and family.
  • Tunnel traffic: Users install the Outline client and use the keys to establish a secure tunnel, bypassing censorship.
