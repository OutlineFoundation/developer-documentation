# Deploy using the Outline Manager

This guide walks you through the process of setting up an Outline Server to
provide secure and unrestricted internet access.

1. Download the Outline Manager for Linux, macOS, or Windows from
   <https://getoutline.org>.

1. In the Outline Manager, click the "+" button to add a new server.

1. Choose your preferred cloud provider (e.g., DigitalOcean, Amazon, Google
   Cloud) or "Set up Outline anywhere" if you have your own server.

1. Follow the on-screen instructions specific to your chosen provider or
   self-setup.

1. The Outline Manager will guide you through creating a server.

## Next steps

Now that your Outline server is set up, you're ready to share access to your
users.
