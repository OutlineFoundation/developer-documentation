Implantar usando um script de instalação

Este guia orienta todo o processo de configuração do servidor do Outline para você ter um acesso seguro e irrestrito à internet.

Pré-requisitos​

  • Um servidor (físico ou virtual) com sistema operacional aceito (Ubuntu 20.04 LTS ou Debian 10)

  • Acesso root ou sudo ao servidor

Instruções​

  1. Faça o download e execute o script de instalação do Outline.
sudo bash -c "$(wget -qO- https://raw.githubusercontent.com/OutlineFoundation/outline-apps/master/server_manager/install_scripts/install_server.sh)"
  1. Siga as instruções na tela.
